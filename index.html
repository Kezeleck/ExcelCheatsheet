An Excel workbook (.xlsx) contains one or more worksheets. Each worksheet is a grid of cells addressed by column letter and row number — A1, BZ47. The grid extends to XFD1048576: 16,384 columns × 1,048,576 rows.

CHAPTER 04

Cells, Rows & Columns

Beginner

Sizing

  • Drag the boundary in the header to resize.
  • Double-click the right boundary of a column header to AutoFit width.
  • Select multiple columns first to AutoFit them all simultaneously.

Merging — and why to avoid it

Merging cells creates layout headaches: you can't sort, filter, or reference merged ranges cleanly. The professional alternative is Center Across Selection: Ctrl+1 → Alignment tab → Horizontal: Center Across Selection. Visually identical to a merged title — but cells remain independent.

Freezing panes

View → Freeze Panes. Excel freezes everything above and left of the active cell.

  • Top row only → click in A2, then Freeze Panes.
  • Top row + first column → click in B2.
  • First three rows + first two columns → click in C4.

Date & Time Functions

BeginnerIntermediate

Excel stores dates as serial numbers — days since 1900-01-00. 1 = 1900-01-01, 45769 = 2025-04-26. Time is the fractional part: 0.5 = noon. This is why date arithmetic just works: =B1-A1→days between

Time arithmetic gotcha

Times over 24 hours
=B1-A1 where the difference is over 24 hours will look wrong because the default time format wraps. Format the result cell as [h]:mm — the brackets tell Excel to show elapsed hours instead of clock hours.

Lookup Functions

IntermediateAdvanced

Lookups are how Excel joins data — pull a related value from one table into another by matching a key. Master XLOOKUP first; learn INDEX/MATCH for legacy files; understand VLOOKUP only because you'll see it.

XLOOKUP — the modern default

XLOOKUP(lookup, lookup_array, return_array, [if_not_found], [match_mode], [search_mode])

  • Looks in any direction — left, right, up, down. No "column index" arithmetic.
  • Built-in fallback. No more wrapping in IFERROR.
  • Exact match by default. (VLOOKUP defaults to approximate, the cause of countless bugs.)
  • Returns an array — perfect for spilling into multiple columns.

VLOOKUP — legacy, still common

VLOOKUP(lookup, table, col_index, [range_lookup]). Looks in the first column of table; returns the value col_index columns to the right.

Always pass FALSE
=VLOOKUP(A1,Tbl,3,FALSE) — always supply FALSE (or 0) as the last argument. The default is TRUE = approximate match, requires sorted data, and returns silently wrong answers when given unsorted data. This has destroyed careers.

XMATCH — position-only lookup

Like MATCH but with the same modern flags as XLOOKUP. Returns the position of a value in an array.

=XMATCH("Q3",B1:E1)→3 =XMATCH(75,A1:A100,1)→first ≥75

CHOOSE & CHOOSEROWS / CHOOSECOLS

CHOOSE(idx, v1, v2, …) picks by index. Modern siblings CHOOSEROWS / CHOOSECOLS pick rows/columns from an array — including by negative indices to count from the end.

=CHOOSE(2,"red","blue","green")→blue

Performance & Scale

Expert

Excel can comfortably handle hundreds of thousands of rows when formulas are written well. It crawls when they aren't.

Volatile functions — recalc on every change

These force the entire dependent chain to recalculate after any change in the workbook. Use sparingly:

  • NOW(), TODAY(), RAND(), RANDBETWEEN()
  • OFFSET(), INDIRECT()
  • CELL(), INFO() with most arguments

Where possible replace OFFSET with INDEX (non-volatile) or use Tables / dynamic-array spill ranges.

Calculation modes

ModeHow to set
AutomaticDefault. Recalc on every change.
Automatic except for data tablesUseful when a Data Table is huge.
ManualFormulas → Calculation Options → Manual. F9 when ready.

Formula speed tips

  1. Avoid full-column references on whole-sheet operations: =SUMIF(A:A,…) scans 1M+ cells. Use Tables or bounded ranges.
  2. Replace nested IFs with IFS / SWITCH / lookup tables.
  3. Use COUNTIFS / SUMIFS instead of array formulas for conditional aggregation.
  4. Compute once, reference many. Use LET to name shared subexpressions.
  5. Helper columns are free. A staged calculation with helpers nearly always outperforms a single massive formula.
  6. Avoid array formulas across full columns — they materialize phantom 1M-row arrays.
  7. Use binary search (XLOOKUP search_mode = 2) for sorted lookups in 100k+ row tables.
  8. Move ETL to Power Query — runs once on refresh, not on every recalc.

File size

  • Convert legacy .xls to .xlsx — significantly smaller and safer.
  • Clear lingering formats beyond the data: Ctrl+End jumps to the "last used cell." If that's miles past your real data, delete the empty rows/cols and save.
  • Compress images: select an image → Picture Format → Compress Pictures → Apply to all.
  • Remove unused styles with the free XLStylesTool (legacy files accumulate them).
  • Pivot caches can balloon — share pivot caches between pivots from the same source.

  5. Show Formulas (Ctrl+`) — see all formulas at once across the sheet.
  6. Watch Window — pin specific cells and watch their values change as you edit.
  7. For VBA: F9 to set a breakpoint, F8 to step, Ctrl+G for the Immediate window. Use Debug.Print liberally.

Defensive formulas

  • Validate inputs: =IF(B2<=0, "Enter a positive number", calc).
  • Use IFNA for lookups so genuine errors still surface — masking with IFERROR can hide real bugs.
  • For division: =IFERROR(a/b, 0) or in the data model, the DAX DIVIDE(a, b, 0).
  • Type-check with ISNUMBER / ISTEXT when reading from external sources.
CHAPTER 28

Expert Tricks

Expert

The hard-won shortcuts that separate fluent users from everyone else.

1 · Multi-cell entry

Select a range, type, and press Ctrl+Enter — the same value or formula goes into every cell. With formulas, references stay relative to each cell.

2 · Quick fill blanks with the value above

  1. Select the column.
  2. F5 → Special → Blanks → OK.
  3. Type = and press ↑.
  4. Ctrl+Enter.

Three seconds. Every blank now references the cell above. Paste-special-values to lock it in.

3 · Tear off a chart from the data

To make a chart that doesn't break when the source moves, select the chart's data series, press F9 in the formula bar — the references become hard-coded arrays. Now the chart is portable.

4 · Camera tool

Customize Quick Access Toolbar → All Commands → Camera. Select a range, click Camera, click elsewhere — you get a live picture of the source. Updates dynamically. Brilliant for dashboards that aggregate views from many sheets.

5 · Custom Lists for sorting

File → Options → Advanced → Edit Custom Lists. Add domain-specific orderings (priority levels, region order, fiscal months). Then sort by them directly.

6 · Format Painter sticky mode

Double-click the Format Painter — it stays active for repeated application. Esc to release.

7 · Quick differences

Select two columns side by side, press Ctrl+\ — selects every cell that differs from the first column in its row. Tag them with a fill color to spot anomalies.

8 · Status bar arithmetic

Right-click the status bar to add Average, Count, Numerical Count, Min, Max, Sum. Select any range and read these instantly without writing a formula.

9 · Roman numerals, Greek, and beyond

=ROMAN(2026)→MMXXVI =ARABIC("MMXXVI")→2026 =UNICHAR(9731)→☃

10 · ADDRESS & INDIRECT for dynamic ranges

ADDRESS(row, col) returns a text address; INDIRECT(text) turns text into a real reference. Useful for sheet-name lookups: =INDIRECT("'"&A1&"'!B2") reads B2 from whichever sheet is named in A1.

11 · Hide a sheet "very hidden"

Right-click sheet tab → Hide only hides — anyone can right-click → Unhide. In the VBE (Alt+F11), select the sheet, set Visible property to xlSheetVeryHidden. Now Unhide doesn't even list it; only VBE can show it again.

12 · Quick chart with F11

Select your data, press F11 — Excel creates a default chart on a new chart sheet. Alt+F1 embeds it on the current sheet.

13 · Convert text-numbers in bulk

Type 1 in an empty cell. Copy it. Select the rogue text-numbers. Paste Special → Multiply. Excel multiplies each by 1, forcing numeric type.

14 · Quick Analysis lens

Select a range — a small icon appears bottom-right. Click it (or Ctrl+Q) for instant access to formatting, charts, totals, tables, sparklines. Underrated.

15 · The N() function

N(value) returns 0 for non-numbers but lets you embed comments inside formulas: =A1+B1+N("includes shipping"). Excel ignores the comment text; humans reading the formula don't.

The most expert habit
Build models that explain themselves. Name your tables and ranges. Use LET for clarity. Add a "How this works" sheet with assumptions and source notes. The best Excel work isn't the cleverest formula — it's the one your colleague can pick up six months later and trust.
